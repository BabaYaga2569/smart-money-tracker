# Design System

## Colors

### Primary Colors
- **Primary Green**: `#00ff99` - Main accent color for buttons, links, and highlights
- **Dark Background**: `#0f0f0f` - Main background color
- **Card Background**: `#1a1a1a` - Component background color

### Status Colors
- **Success**: `#00ff99` - Positive actions and success states
- **Warning**: `#ffcc00` - Warning states and alerts
- **Danger**: `#ff4444` - Error states and destructive actions
- **Info**: `#00ccff` - Informational messages

### Text Colors
- **Primary Text**: `#ffffff` - Main text color
- **Secondary Text**: `#b0b0b0` - Secondary text color
- **Muted Text**: `#808080` - Disabled or muted text

## Typography

### Font Families
- **Primary**: System font stack (Arial, Helvetica, sans-serif)
- **Monospace**: For code and numbers

### Font Sizes
- **Extra Large**: `2rem` - Page titles
- **Large**: `1.5rem` - Section headers
- **Medium**: `1.25rem` - Card titles
- **Base**: `1rem` - Body text
- **Small**: `0.875rem` - Helper text
- **Extra Small**: `0.75rem` - Labels and captions

## Spacing

### Scale
- **Extra Small**: `0.25rem` (4px)
- **Small**: `0.5rem` (8px)
- **Medium**: `1rem` (16px)
- **Large**: `1.5rem` (24px)
- **Extra Large**: `2rem` (32px)

## Border Radius
- **Small**: `8px` - Buttons, inputs
- **Medium**: `12px` - Cards, modals
- **Large**: `16px` - Large containers

## Shadows
- **Card Shadow**: `0 8px 20px rgba(0,255,153,0.1)`
- **Hover Shadow**: `0 12px 24px rgba(0,255,153,0.15)`

## Layout

### Container Widths
- **Max Content Width**: `1200px`
- **Card Max Width**: `400px`

### Breakpoints
- **Mobile**: `< 768px`
- **Tablet**: `768px - 1024px`
- **Desktop**: `> 1024px`
