## 5. Size Comparison

### Banner Dimensions

**BEFORE:**
- Height: ~56px (12px + 12px padding + 16px font + line-height)
- Font: 16px, weight 600
- Padding: 12px 24px

**AFTER:**
- Height: ~41px (8px + 8px padding + 13px font + line-height)
- Font: 13px, weight 500
- Padding: 8px 16px

**Reduction:**
- 27% smaller in height
- 33% less padding
- 19% smaller font
